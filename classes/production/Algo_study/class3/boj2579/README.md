### :octocat: Taekyung2
![image](https://user-images.githubusercontent.com/37056992/96330686-3aa39200-1092-11eb-9426-004d7a817fb3.png)

- i번 째 계단을 밟았을 때 점수는
- i-2번째 계단을 밟았을 때 점수 + i계단 점수와
- i-3번째 계단을 밟았을 때 점수 + i-1계단 점수 + i계단 점수 중에 큰 점수
- 중복 계산을 막기위해 메모이제이션
